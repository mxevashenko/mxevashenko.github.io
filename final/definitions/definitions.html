Definitions of common Compuer Programming words

Rough Draft PDF Version

func•tion(noun)

A sequence of instructions that preform a task, packaged as a unit.

"I wrote a function to print text."

in•te•ger(noun)

A whole number that is not a fraction.

"An integer does not have a decimal point."

string(noun)

An array of characters or text.

"You must convert a string before preforming math on it."

syn•tax(noun)

A set of rules that define the structure of a language.

"C# syntax requres a type specification."

com•pil•er(noun)

A program that translates instructions into machine code or lower-level form to be executed by a computer.

"The compiler threw an error."
